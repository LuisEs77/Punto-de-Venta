# Pruebas Unitarias del Proyecto Punto de Venta

Este repositorio contiene únicamente los archivos de pruebas unitarias extraídos del proyecto original.

## Estructura de carpetas

- `src/dtos/__tests__` — Pruebas unitarias para los DTOs
- `src/controllers/__tests__` — Pruebas unitarias para los controladores
- `src/services/__tests__` — Pruebas unitarias para los servicios
- `src/guards/__tests__` — Pruebas unitarias para los guards
- `src/auth/__tests__` — Pruebas unitarias para autenticación
- `src/errors/__tests__` — Pruebas unitarias para manejo de errores

## Ejecución de pruebas

Estas pruebas están diseñadas para ejecutarse con Jest. Para correrlas:

1. Instala las dependencias necesarias (por ejemplo, Jest y TypeScript):
   ```bash
   npm install
   ```
2. Ejecuta las pruebas:
   ```bash
   npm test
   ```

## Notas
- Este repositorio no contiene código fuente de la aplicación, solo los archivos de pruebas unitarias.
- Si necesitas el código fuente completo, consulta el repositorio original.
